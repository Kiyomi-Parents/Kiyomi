# BSBot

cool bot

### Install

    # Install requirements  
	python3 -m pip install -r requirements.txt  

### Usage

    # Set discord token  
	export DISCORD_TOKEN=<My very secret token>  
  
	# Start bot  
	python3 BSBot.py  


### Tests
    # Install tools
    python3 -m pip install pytest pytest-xdist
    
    # Run tests
    pytest -v --numprocesses=auto
